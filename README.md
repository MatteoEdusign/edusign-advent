A beautiful, single-page Advent Calendar web application designed for Edusign.

## Features
- **Custom Design**: Edusign branding with Deep Dark Teal and Mustard Yellow.
- **Interactive**: 24 unlockable days with fun facts, tools, and tips.
- **Animations**: Falling snow, twinkling stars, and a flying Santa sleigh!
- **Easter Eggs**: Hidden developer messages and more.

## Tech Stack
- HTML5
- CSS3 (Glassmorphism, Animations)
- Vanilla JavaScript

## Setup
Simply open `index.html` in your browser to view the calendar.

## Deployment
Ready to be deployed on Vercel.
